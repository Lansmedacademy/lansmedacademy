ASSETS FOLDER
=============
Use this folder for downloadable and static assets as the platform grows, e.g.:
  - PDF study notes            (assets/notes/)
  - Medical illustration packs (assets/illustrations/)
  - Brand assets / press kit    (assets/brand/)
  - Fonts hosted locally        (assets/fonts/)

Reference them in your HTML like:  href="assets/notes/cardiac-cycle.pdf"
